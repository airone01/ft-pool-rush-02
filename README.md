### `make` commands

#### Compiling

To compile use

```bash
make
```

## Cleaning

To clean use

```bash
make clean
```

And to clean everything, including the compiled binaries, use

```bash
make fclean
```

#### Re-compiling

To remove old files and then re-compile use

```bash
make re
```
